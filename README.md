# FlashMind

FlashMind is a dynamic and interactive Flashcard Quiz Application built with Flutter. It helps users efficiently learn and memorize information through customizable flashcards.

## Features

*   **Subject Management**: Organize flashcards into different subjects or decks.
*   **Interactive Flashcards**: Tap to flip between question and answer.
*   **Progress Tracking**: Mark cards as 'Learned' or 'Forgot' to track your learning progress.
*   **Dynamic UI**: Modern, animated user interface built with Flutter.
*   **Confetti Animations**: Celebrate when you successfully complete a deck!

## Getting Started

To run this project locally, ensure you have Flutter installed.

1. Clone the repository:
   ```bash
   git clone https://github.com/shabareesh390/CodeAlpha_FlashcardQuizApp.git
   ```
2. Navigate into the directory:
   ```bash
   cd CodeAlpha_FlashcardQuizApp
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
